# 写作手法库

## 当前风格技能

### 爽点制造手法
- **反差萌爽点**：煤球在外是高傲S级异能兽，在家是蹭头撒娇的猫
- **打脸爽点**：看不起陆野的人被煤球暗中出手打脸
- **炫技爽点**：异能战斗场面，煤球或陆野展现强大实力
- **感情线小甜**：煤球助攻陆野与暗恋对象互动
- **现实小翻盘**：煤球用极克制方式解决外卖差评、手机电量、临时工作、房租/水电等现实压力，不暴露真实身份
- **普通人反杀**：陆野利用外卖路线、街景图、监控盲区、生活常识和假线索误导追猎者，证明他不是只会被保护

### 节奏控制
- 慢档（铺垫章节）：日常互动、感情线推进、煤球卖萌，每3-4章至少1章
- 中档（升温章节）：异能事件初现、次要反派登场，占60%+
- 快档（突破章节）：主线重大推进、煤球实力恢复关键节点，每卷不超过3次

### 对话技巧
- 吐槽役：陆野的内心吐槽与嘴上吐槽
- 动作互动：煤球不能说完整人话时，用尾巴、爪子、眼神、装猫失败和被迫配合制造互动密度
- 傲娇对话：煤球的嘴硬心软可转化为动作回应，不强求口头对白
- 异能者之间的黑话：增加世界真实感

### 章末钩子类型
- 悬念型：暗示接下来有大事发生
- 反转型：本章结尾抛出一个意外信息
- 期待型：预告下一章的爽点
- 好奇型：埋下一个新的谜团
