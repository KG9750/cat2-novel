# 更新记忆 - 第15章

## 章节摘要
第15章：蓝夹克在办公室告诉陆野两件事——①追猎者面包车撤到省道收费站外（未放弃，换班盯梢）；②煤球检测仪重启前缓存的波形在系统里有三年前类似频率记录，但对应卷宗被人抽走，只留下借阅章编号——章编号与煤球频率特征码共享同一段前缀。蓝夹克暗示"抽走档案的人可能还在系统里"。随后蓝夹克安排陆野送一批常规动物健康周报到市区瑞鹏动物医院（合作机构），收件人恰好是沈知夏。陆野带煤球乘公交前往。沈知夏首次现实登场——白大褂、马尾、兽医助理。她专业审视煤球（行为评估而非看可爱），说"它不像流浪猫"。摸煤球头时指尖感觉到异常（温度/纹理），皱眉但未追问。她给陆野名片——背面手写"动物行为观察记录（个人）"+邮箱。煤球让摸了一下就缩回去，用尾巴拍陆野手背表示"欠人情"。
