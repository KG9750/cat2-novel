# 更新记忆 - 第12章

## 章节摘要
第12章：追猎者灰衬衫使用车载全频脉冲远程唤醒珠子，锁定陆野和煤球在荒草地。两架无人机逼近——第一架近距离探测，第二架在远处记录频率数据（追猎者真正目的：测煤球的能力波形）。煤球被迫出手——脊背沿脊柱透出幽蓝光，释放电磁/气浪脉冲干扰无人机（第一架失控迫降、第二架远距录像）。出手短促（数秒）但消耗极大——煤球体温升高、呼吸变浅。追猎者面包车从东侧绕上省道逼近公交站。危急时刻，一辆白色防疫车（临江市动物疫病预防控制中心）从南边出现。车上下来两人：白衣防护服持金属箱、蓝夹克（异能局外勤）。蓝夹克要求"把猫放进笼子，先做D级体检"。北边面包车在倒车退让——追猎者与白车不是一伙，但认可白车权限更高。陆野确认对方非追猎者后，带煤球自愿跟白车走。

## 新增信息
- **煤球异能展示**：沿脊柱释放幽蓝光脉冲——电磁干扰+气浪——可导致无人机失控/电路干扰。展示级别接近D级全力，短促高强度。出手后体温升高、呼吸急促——消耗极大。
- **追猎者真实目的**：不全是抓捕——第二架无人机远距离录像/录频率。灰衬衫的目标是煤球的能力波形数据。每次压迫都在逼煤球暴露更多参数。
- **异能局首次出场**：白色防疫车（动物疫病预防控制中心外壳）、蓝夹克外勤、白衣持金属箱（可能是检测设备）。非敌意——要求D级体检而非强制收容。
- **权限关系**：追猎者在白车面前退让——异能局权限高于追猎者，但两方互相认识。
