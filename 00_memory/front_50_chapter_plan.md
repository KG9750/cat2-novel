# 前50章详细规划

> 作用：作为第9章起的中程执行蓝图。写作时仍以 `novel_plan.md` 的主线红线、`novel_state.md` 的即时状态和每章门禁为准；本文件负责把前50章的节奏、爽点、伏笔回收和人物推进拆到章级。

## 总体原则

- **题材校准**：都市异能 + 萌宠日常 + 轻松爽文。第4-8章已经偏悬疑压迫，后续必须定期回到现实小爽点和煤球反差萌。
- **节奏配额**：每章最多触发 A/B/C 中一项。快档后必须接慢档或中档缓冲。前50章内不解决煤球重伤真相、不揭完整宿敌身份、不让陆野越级成为强者。
- **主角成长**：陆野前50章的核心价值不是战力，而是普通人的路感、观察力、低成本求生和对煤球的选择。
- **煤球边界**：第1-50章维持"重伤D级逐步恢复到接近C级"。可以短促爆发，不能长期碾压。
- **感情线策略**：第9章埋沈知夏种子；第20章前正式出现；前50章只做到"可互相信任的现实线支点"，不早甜、不告白。沈知夏必须有自己的私人动机：她追查三年前动物异常/导师旧案，不是单纯为陆野冒险。
- **机构策略**：临江异能局不是纯反派。前50章要写出外勤派、保守派和疑似内鬼的裂缝，为第50章"被迫站队"服务。
- **因果桥策略**：从追杀/收容/约谈切回日常时，必须给出权限、监控、临时任务或现实身份理由；不靠巧合让角色自由行动，也不靠会议硬推下一章。
- **章内回报策略**：每章都要给读者一个可感知的小回报，可以是陆野赢一个程序点、煤球反差萌兜底、现实困境缓解、沈知夏发现证据，或主宠配合打出一次低成本胜利。
