# 小说主线计划

> 本文件是长期记忆的核心，防止剧情偏离主线。每次写作前必读。
> 修改规则：仅在用户明确同意后修改。

## 核心设定（不可偏离）
- 书名：《我捡的流浪猫，竟是S级异能兽》
- 类型：都市异能 + 萌宠日常 + 轻松爽文
- 核心卖点（一句话）：社畜捡到S级异能兽，猫要报恩让他平步青云
- 预计总字数：50-80万字
- 终极结局：陆野与煤球并肩作战，揭露异能局高层与暗势力的勾结，成为异能界新秩序缔造者。煤球恢复巅峰实力后选择留在陆野身边。陆野事业爱情双丰收。

## 核心人物弧线
- 主角陆野：三流大学毕业生找不到工作 → 在煤球帮助下进入异能界 → 逐步觉醒成长为A级异能者 → 异能界新秩序缔造者
- 煤球（S级异能兽）：重伤濒死D级幼猫 → 逐步恢复实力 → 揭开心结与过往 → 恢复S级巅峰 → 选择留在陆野身边
- 暗恋对象（待命名）：陆野暗恋的女孩 → 与陆野关系在煤球助攻下逐步升温 → 发现异能世界 → 成为陆野的坚实后盾
- 宿敌（待命名）：煤球的旧识 → 暗势力首领 → 最终被击败

## 力量体系阶梯
F级（微量异能感知） → E级（异能初步觉醒） → D级（异能初级运用） → C级（异能实战化） → B级（异能战斗形态） → A级（异能领域展开） → S级（异能法则级）

## 绝对规则（红线）
1. 煤球永远不会背叛陆野（报恩是起点，羁绊是终点）
2. 猫咪不会死，也不会被严重永久伤害
3. 主角不能长期被虐——可以受压但必须有翻盘点
4. 非终局章节禁止最终解决主线核心冲突
5. 爽点必须每章都有——可以是反差萌、打脸、异能炫技或感情线小进展
6. 每章结尾必须有强吸引力钩子（悬念/反转/期待/好奇），让读者不翻下一章就难受
