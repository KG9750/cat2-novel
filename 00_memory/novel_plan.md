## 核心设定（不可偏离）
- 书名：《我捡的流浪猫，竟是S级异能兽》
- 类型：都市异能 + 萌宠日常 + 轻松爽文
- 核心卖点（一句话）：社畜捡到S级异能兽，猫要报恩让他平步青云
- 预计总字数：50-80万字
- 终极结局：陆野与煤球并肩作战，揭露异能局高层与暗势力的勾结，成为异能界新秩序缔造者。煤球恢复巅峰实力后选择留在陆野身边。陆野事业爱情双丰收。

## 核心人物弧线
- 主角陆野：三流大学毕业生找不到工作 → 在煤球帮助下进入异能界 → 逐步觉醒成长为A级异能者 → 异能界新秩序缔造者
- 煤球（S级异能兽）：重伤濒死D级幼猫 → 逐步恢复实力 → 揭开心结与过往 → 恢复S级巅峰 → 选择留在陆野身边
- 沈知夏：陆野暗恋对象 → 先以旧聊天/订单种子进入 → 动物相关专业观察支点 → 因三年前动物异常/导师旧案拥有私人调查动机 → 与陆野关系在煤球助攻下逐步升温 → 发现异能世界 → 成为陆野的坚实后盾
- 宿敌（待命名）：煤球的旧识 → 暗势力首领 → 最终被击败

## 第一卷第9-12章执行细纲（用户已确认）
- **总体节奏**：第9-10章降压，写"短暂安全区里的萌宠/求生/小爽点"，让读者从第4-8章连续追踪压迫中喘一口气；第11-12章再升温到第一次异能事件。
- **第9章方向**：地下室暂时安全。陆野先不打开金属盒，而是处理吃饭、手机电量、外卖账号、房租/水电等现实问题。煤球用极克制的微量异能帮他解决一个小现实困境，但必须装成普通猫，制造反差萌。第9章开始埋暗恋对象种子：手机旧聊天框、外卖订单收件人、招聘联系人或朋友圈头像任选其一，只种下名字/印象，不正式展开感情线。
- **第10章方向**：继续安全区求生。陆野利用普通人经验整理地下室、处理气味/脚印/路线，煤球高傲指挥却被迫配合"装猫"细节。安排一个小爽点：煤球暗中帮陆野拿到临时工作机会、保住外卖账号、解决差评/赔付或缓解房租压力。
- **第11章方向**：追猎者的搜索逼近旧工业区。陆野不用异能，靠外卖骑手路线、监控盲区、街景图和假线索反杀一次追踪者，让对方误判方向；这是普通人脑子的第一次明确胜利。
- **第12章方向**：第一次异能事件爆发。煤球被迫小范围出手，但不完整坦白身份；章末钩子引向第二卷"真相初现"。
- **硬要求**：第9-12章至少两个小爽点；每章保留萌宠反差；不提前完整揭示煤球真名、金属盒内容物、笔记作者身份和S级完整过往。

## 力量体系阶梯
F级（微量异能感知） → E级（异能初步觉醒） → D级（异能初级运用） → C级（异能实战化） → B级（异能战斗形态） → A级（异能领域展开） → S级（异能法则级）

## 绝对规则（红线）
1. 煤球永远不会背叛陆野（报恩是起点，羁绊是终点）
2. 猫咪不会死，也不会被严重永久伤害
3. 主角不能长期被虐——可以受压但必须有翻盘点
4. 非终局章节禁止最终解决主线核心冲突
5. 爽点必须每章都有——可以是反差萌、打脸、异能炫技或感情线小进展
6. 每章结尾必须有强吸引力钩子（悬念/反转/期待/好奇），让读者不翻下一章就难受
