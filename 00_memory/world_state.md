# 世界观状态

> 记录已揭示和隐藏的世界设定。

## 已揭示设定（读者已知）

### 表层世界
- 时间：2026年前后，当代都市
- 地点：临江市（虚构），一座普通的大城市
- 社会状态：普通人完全不知道异能世界的存在

### 异能世界（里层）
- 异能者：拥有特殊能力的人类，从F级到S级
- 异兽：拥有异能的动物，同等分级
- 异能局：官方异能管理机构（具体架构待揭示）
- 协战观察：陆野第50章后进入的受限协作身份；有住处、报备、监控、任务补贴和证据查阅限制，观察在前，协战在后
- 特别项目临江组：第54章通过补贴备注露出的旧名；蓝夹克判断三年前后不该再出现在新账目里
- 暗势力：黑环（前50章逐步确认其名），与异能局对立，目标与煤球过去和三弧纹章有关
- 黑环视觉标记：第17章先以黑色环状划痕/贴纸等形式出现，第21章再正式命名为"黑环"
- 三弧纹章：已在卡片、地下室天花板和金属盒底部出现。读者已知它不是普通标记，而是能产生共鸣的"通道"
- 卡片共振追踪：低功率激活信号扫过全城时，带纹章的卡片会响应；煤球降到D级后仍保留S级共振频率基准线
- 旧工业区地下室：煤球旧安全点之一，存在干草窝、草药盆、天花板纹章和隐藏隔间

- 主楼/旧地下层：特别项目时期的内部代称，三年前撤离命令与“黑猫不能进主楼”警告均指向此处；近期有合法维护权限读取门禁。
- 共鸣型危险感知：陆野复测结论，不是稳定觉醒异能；与煤球同场时增强，离开煤球后下降但不归零，可训练到短暂三秒路线判断。
- 封印频率：煤球体内存在一段不属于自身的稳定频率，可能同时具备保护和压制作用；半解封会带来恢复前兆，也会增加被黑环定位风险。
- 鸦医：黑环中层代号，擅长处理异兽频率，曾拥有或使用过异能局临时医疗权限。
- 特别项目重启：近期以“历史高危资产复核与残频清理预案”名义出现，申请目的需要S级异兽残频。

- 临江变量链：第91章审议组正式命名，包含陆野、煤球、沈知夏和外勤监管；第100章确认具备特殊协战资格。
- 特殊协战资格：第100章主楼复核任务后获得，代表陆野有资格进入更高层异能局棋盘，但伴随更高监管。
- 官方应急频道污染：第100章黑环借异能局官方应急频道发动突袭，证明内鬼或旧权限深入官方应急系统。

## 设定规则
- 异能者觉醒需要触发条件（濒死/极强情绪/天赋觉醒），不可无理由觉醒
- 异兽无法主动对人类说出完整语言（但可通过意念/动作传达）
- 异能战斗的物理痕迹可以被异能局用特殊手段消除
- 煤球当前必须极克制使用能力；第9-10章只能用微量异能解决现实小困境，不可直接碾压追猎者
