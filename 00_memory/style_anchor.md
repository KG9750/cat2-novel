# 风格锚点

> 每章写作前参考，校准文风基线。每10章深度更新一次。

## 当前风格基调
- 类型：轻松搞笑 + 都市异能 + 萌宠
- 叙事腔调：第一人称或第三人称有限视角，带吐槽口吻
- 互动密度：50%+（陆野吐槽、煤球动作回应、误解/反应/肢体交流都计入；煤球不能说完整人话时，不强求对白占比）
- 句长：中短句为主，动作场景短句连击
- 节奏：每章一个小爽点，每3-5章一个中等爽点
- 基调红线：可紧张不可压抑，可有悬念不可虐主

## 参考风格
- 目标平台风格：番茄小说轻松向爽文
- 互动风格：日常化、有梗、吐槽自然；用动作回应制造"高傲异能兽被迫装猫"的反差萌
- 战斗描写：异能炫技型，不血腥
- 萌宠互动：暖+反差+傲娇对话

## 禁用写法
- 大段心理独白（单段不超过150字）
- 文绉绉的书面语对白
- 过度环境和风景描写（每场景不超过200字）
- 翻译腔句式
- 角色以作者口吻解说世界观（通过对话和行动自然展示）
