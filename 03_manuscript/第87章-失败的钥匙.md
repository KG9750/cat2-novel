# 第八十七章 失败的钥匙

煤球一整晚没有睡。

它蹲在旧宿舍窗台上，面向收容点后院。院子里只有一盏路灯亮着，灯罩上积了半年的灰，光打下来是昏黄的，落在煤球背上，黑毛边缘泛出很淡的蓝——像一层没完全收回去的旧伤。夜风从窗缝灌进来，煤球的耳朵转了半圈，朝向后院铁门的方向，像在确认没有新的频率靠近。

它的呼吸比平时浅。胸口的起伏几乎看不见。尾巴从窗台边缘垂下去，一动不动，尾尖离地还有三厘米。整只猫像一座被夜露打湿的石像。

陆野也没睡。

他坐在床边，手里拿着沈知夏给的备用体征贴。贴上去，撕下来，再贴上去。贴了七次，撕了七次。手腕内侧的刺痛已经退了，但那种针尖刮过血管的感觉还残留在神经末梢里，像一首歌被关了以后耳朵里还响着前奏。右手小指到手腕那一小段皮肤，触觉比平时钝了半拍，指尖摸体征贴的塑料包装时，能感觉到光滑，但感觉不到凉。

沈知夏说他今晚最好休息。窗台上那只猫不睡，他也睡不着。

宿舍很安静。热水器下午终于修好了——维修单排了整整三周，今天中午小周在走廊里喊了一句"热水器的零件到了"，陆野差点给他鼓掌。冲了今年第一个真正的热水澡之后，现在水龙头偶尔滴一滴，打在瓷砖上，声音比心跳还慢。

走廊里有脚步声响了一下。蓝夹克的靴子，陆野认得那个节奏。脚步在门外停了停，没有敲门。停了大概五秒，又走了。外勤队长知道今晚不该打扰。

鸦医走之前说的那四个字，整个收容点都听见了。失败的钥匙。秦主任把这句话写进了安全日志。小周在系统里搜了一圈，没搜到"失败的钥匙"这个关键词在任何旧档案里的记录。它不像被删过，更像从来就没被录进系统。有人刻意没把它写成文字。

陆野第三次把体征贴撕下来的时候，开了口。

"你要是想咬点什么，"他说，"新猫包编号牌可以给你。"

煤球没理他。耳朵都没有转。

"但别咬报告。报告要归档。"

煤球尾巴动了一下。仅仅一下。尾尖从窗台边缘抬起来，又落回去。

陆野知道它在想什么。失败的钥匙。鸦医说这四个字的时候，煤球的瞳孔缩了一下——某种比愤怒更旧的东西被翻了出来。像一个人在街上忽然听到二十年前班主任叫自己的名字，那种本能回头。那层害怕之下，是身体还记着。

而最让陆野在意的是鸦医的语气。不是嘲讽。不是威胁。说出来的时候像在报一个病历上的诊断结论。

他把金属盒从床头柜拿下来，放到窗台下方。盒子在昏暗里泛着银灰色冷光，盒盖缝隙里透出的温度比室温略高一点。它今晚格外安静——平时偶尔会自己热一下，今晚却像在等什么。

"不说也行。你现在不欠任何人解释。"

煤球终于回头看他。

那一眼很长。瞳孔里的蓝光没有扩散，只是静静地亮着，像一盏忘了关的小夜灯。陆野见过煤球很多种眼神——嫌弃、护食、忍痛、高傲、昏昏欲睡时半睁着眼的敷衍——但这一种他没见过。一只猫在衡量一个人类能不能接住它接下来要做的事。它在窗台上蹲了将近四个小时，盘算的只有一件事：把这块碎片拿出来，值不值得。

然后它从窗台跳下来。

落地几乎没有声音。它走到金属盒旁边，绕着盒子转了半圈。尾巴从盒盖缝隙上方扫过，动作很轻，反复了两次。第一次扫过去，盒缝里没有任何反应。第二次扫过去，盒缝边缘微微亮了一下，是暗蓝色的，一闪就灭。

煤球确认了那道旧疤还在。

它伸出右前爪，用爪垫轻轻按了一下盒缝。

金属盒的缝隙本来只够漏出一点声音。上次它自己发热，盒盖边缘翘起了不到一毫米，被秦主任用审议组封条压了回去。封条还在，但胶已经有点松了——盒子的温度在这段时间里反复变化，冷热交替，胶慢慢失去了黏性。审议组的红色封条纸上，"秦"字的边缘已经翘起了半个角。

煤球这一按，缝隙慢慢扩大了半毫米。

陆野屏住呼吸。

没有蓝光喷出来。没有旧案真相自动播放。没有谁的声音从盒子里传出来。盒子里只有很轻的一阵嗡声，像老冰箱压缩机启动前那一刻。

里面露出一枚很小的碎片。

指甲盖大小，旧钥匙形状。表面有灼烧痕迹——不是火焰烧的，更像高温消毒留下的焦痕，边缘整齐。碎片上刻着半截三弧纹章，三条弧线从中间断开，断裂处恰好和金属盒底部的刻痕能对上，像一个签名从中间被撕成两半。

陆野没有伸手拿。

煤球看着他。

然后他明白了。这不是答案。它在窗台上蹲了一整晚，盘算的也只有一件事：要不要把这块碎片拿出来。漆黑的猫蹲在路灯下四个小时，从头到尾只做了一个决定：把旧钥匙推出来半毫米。

鸦医叫它"失败的钥匙"。钥匙。失败的。这个人知道它的过去，知道它曾经是用来打开什么的东西——或者是用来锁住什么的东西。而这块碎片，就是那段过去里它唯一愿意拿出来的一小片。

陆野把视线从碎片上移开，看回煤球的眼睛。

"你以前，是钥匙？"

煤球没有点头，没有摇头。它把爪子从盒缝上移开，爪子在空中悬了一下，然后轻轻放在碎片旁边。没有碰。只是放在同一个平面上。

像在说：我解释不了。但你可以看。

陆野在那只爪子旁边看到了很细的旧伤。藏在肉垫边缘，被黑毛遮住了大半。那不像打架留下的撕裂伤，更像规整的愈合线——手术切口，或者某种实验留下的组织取样痕迹，愈合得比自然伤口平整得多。至少三道，排得很均匀。

他没有问那是什么。他只是看了两秒，然后把视线移回煤球的眼睛。

"好。"他说，"我看见了。"

煤球收回爪子，重新跳上窗台。这次它的尾巴没有贴着身体。尾巴垂下来，尾尖轻轻晃了一下——四个小时以来第一次。

陆野把那零点五毫米的盒缝重新合上。封条的胶已经失效了，他从床头柜抽屉里找出一截布基胶带重新贴了一道。贴完才发现胶带是蓝色的，和金属盒的银灰色完全不搭，远看像车祸现场的临时包扎。

煤球看着他贴胶带，耳朵转了一下。右耳往前，左耳往后——它在表达一种混合评价。

"凑合用。"陆野说。

第二天早上，沈知夏来的时候带了早饭。豆浆、馒头、两个茶叶蛋。她把袋子放在桌上，先看了一眼煤球。煤球蹲在窗台上，尾巴垂下来，姿势比昨晚松了一点，但眼睛下面那圈毛色有点暗——陆野以前以为黑猫不会有黑眼圈，现在知道了，是有的。

"它一晚没睡？"

"我也一晚没睡。"

沈知夏翻开记录本，在煤球的体征栏写了几笔。她没有写频率数据，写的是行为观察：夜间警戒姿势持续时间长，无明显攻击预备，应激级别中等。然后她抬头看陆野。她什么都没问——没问"失败的钥匙"是什么意思，没问鸦医为什么认识煤球，没问他昨晚为什么不去追问。她看到了桌上的布基胶带和金属盒，看到了煤球尾巴垂下来的角度和昨晚不同。

她只是把豆浆推到他手边。

"趁热。"

陆野端起豆浆，喝了一口。烫。舌尖被烫得发麻，但他没吭声。沈知夏在观察他——他知道她在观察他。她在看他有没有掩饰疼痛，有没有用废话转移话题，有没有把手指藏进口袋里。

他把杯子放下，手指放在桌面上，摊平。

"盒子里有一块碎片。"他说，"钥匙形状，纹章刻痕。煤球昨天晚上拿出来的。"

煤球从窗台上跳下来。它走到沈知夏面前，看看她，又看看金属盒。没有绕圈子，没有犹豫。

沈知夏放下笔，蹲下来和煤球平视。

"你要给我看什么？"

煤球用爪子轻轻推了一下金属盒。它没有按盒缝，而是把整个盒子推到了沈知夏面前。然后它伸出爪子，把碎片从盒缝里拨出来一点，往沈知夏的方向推了半寸。

它的爪子很稳。比昨天在档案室里按住猫包边缘的时候稳得多。推完之后，它收回爪子，跳到椅背上，把尾巴绕到前爪旁边，低下头，不看任何人的眼睛。

沈知夏看着那块碎片，沉默了很久。她认出了那半截三弧纹章——和导师旧记录页脚上的纹章碎片照能对上，和样本箱上的残码也能对上。纹章的另一半，原来一直藏在煤球守着的金属盒里。

然后她在记录本上写下了一行观测记录。字迹比平时重，笔尖在纸面上压出了浅浅的凹痕。

这件事已经越过了纹章和频率本身。一只猫愿意把伤口从盒缝里推出来，推到一个人类和另一个人面前。

陆野把剩下的半杯豆浆喝完。舌头还是麻的，但他觉得没关系。

窗外的路灯在天亮之前已经熄了。临江的晨光从窗缝里挤进来，落在煤球闭着的眼睛上。它的耳朵仍然朝着金属盒的方向。
