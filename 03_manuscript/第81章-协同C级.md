# 第八十一章 协同C级

地下库一战的复盘会，摆在第二天上午九点。

会议室冷气开得太足。长桌一头坐着蓝夹克，另一头蹲着一只黑猫。煤球面前放了一份纸质文件，封面印着两行字：「异常陪护兽临时表现评估」。煤球盯着「陪护兽」三个字，瞳孔微微收窄。它没有拍桌——爪子悬在纸面上方两厘米的位置，停住，然后收回去。

陆野低声说：「至少没用那个字。」

煤球偏头看他。

「宠物。」陆野说，「至少没写宠物。」

煤球尾巴扫过文件边角，把封面扫歪了四十五度。

蓝夹克没有纠正。他把地下库的行动路线投到白板上。封地漏、剪管线、保护记录箱、压制灰雾——每一步都附了时间戳和对应人员。以前陆野的名字常年出现在「协助说明」或「观察备注」栏里，字体比正文小一号。今天不一样。系统单独给他划了一列。

列名很短：协同路线判断。

列内容更短：有效。

蓝夹克翻到下一页：「临时等级，协同C级观察对象。」

陆野看着「观察对象」四个字，第一反应不是激动。他当过外卖骑手、补习班临时工、各种系统的临时用户，「临时」和「观察」这两个词放在一起，他太熟了——意思是可以用，但还在看。

「观察对象能不能删？」他问。

「现在删不了。」蓝夹克说。

「那协同C级有什么实际内容？」

小周替他答了：「补贴标准上调一级，装备升级，任务权限从外围协查升到随队协同。」

他把三样东西推到陆野面前。一个黑色腕带，内侧贴了体征感应片。一张新证件，照片是他入职时拍的，表情像刚睡醒。还有一只新猫包，编号印在侧面。

煤球看见那只猫包，耳朵不抖了。

编号：XH-CAT-C-临江-001。

临江异能局，协同C级，第一只正式编号的动物。

陆野拿起猫包看了看，诚恳地说：「你现在有工号了。」

煤球一爪按住猫包边缘。力度不轻不重，恰好把包按得翻了个面，编号朝下。

蓝夹克没有理会这场小型抗议。他继续往下翻，列出具体待遇。补贴数额比之前翻了一倍，外勤食堂权限从访客级提到二级，旧宿舍热水器的维修单被划进了优先队列。陆野看着那张维修单，觉得自己这辈子第一次靠异能世界解决了热水问题。冬天洗冷水澡这件事，他已经忍了大半个临江的雨季。

现实好处一条条落在屏幕上，会议室里的气氛却没有跟着变轻。

保守派代表把手里的笔放下，声音不高：「待遇可以先按临时标准走，但评价不能只看一次地下库行动。协同C级——与其说是正式能力等级，不如说是对人和异常兽共同作用结果的临时标注。换句话说，陆野本人是否具备可控价值，还没有结论。」

这句话说得很平，像在讲制度。陆野却听出了里面的刀口。

「你的意思是，」他问，「好用的时候叫协同，不好用的时候叫依赖？」

保守派代表抬眼：「我说的是风险归属。」

沈知夏坐在旁听席最末端，低头写了一行字。她没有插话，只把地下库那晚的体征记录推给小周。小周接过去扫了一眼，补进投影附件里。

附件页面展开：陆野下达路线指令时，煤球B级前兆没有扩大；沈知夏记录箱受保护期间，煤球影态稳定在门口三米范围；灰雾压制后，陆野心率峰值比煤球频率峰值慢半拍。

「风险归属要看完整链路。」沈知夏这才开口，「如果只看煤球影态，陆野像依赖异常兽；如果看路线指令、灰雾落点和记录箱保全，陆野是在压住异常兽扩散范围。他的价值不在战斗，在提前把混乱拆成能处理的几段。」

会议室安静了两秒。

煤球把尾巴从猫包编号上挪开，尾尖轻轻点了一下桌面。

像给这份解释盖了个猫爪章。

蓝夹克趁这两秒把话接住：「所以临时标注叫协同C级，不叫陆野C级，也不叫煤球C级。它标的是这条链条：陆野判断路线，煤球提供关键压制，沈知夏保留证据，外勤执行封控。链条能用，才有后续任务资格。」

陆野第一次觉得「协同」两个字没有那么虚。它既非给他贴金，也非给煤球降格，只是把那晚所有人的分工写进了同一个系统格子里。

但格子写得越清楚，后面的监管也越容易套上来。

小周把腕带启动，屏幕上跳出一串权限说明。任务地图可以提前三分钟加载，紧急撤离路线可以同步到外勤频道，体征异常会自动上报医疗组。陆野看着这些功能，脑子里却同时浮出另一面：提前加载意味着行踪会被记录，撤离路线同步意味着每一次判断都会留痕，体征上报意味着他连什么时候心跳快一点都要被系统看见。

「这玩意能关吗？」他问。

小周很诚实：「能关，但关了就不能出勤。」

陆野把腕带扣上。黑色腕带贴住手腕内侧，凉得像一圈刚拧紧的水管卡箍。

煤球跳到桌上，用爪尖拨了一下腕带。拨完又去拨自己的新猫牌。猫牌是哑光黑，边缘刻着临江异能局的简化标识，背面有一枚很小的定位扣。煤球盯着定位扣看了两秒，抬头看陆野。

陆野立刻说：「这个不是我申请的。」

蓝夹克补刀：「但你签收了。」

煤球把猫牌推到陆野面前，动作慢而坚定。

会议室里有人忍笑。保守派代表没笑。他在记录纸上写了一行：异常兽对监管设备有排斥反应。

沈知夏隔着桌面看见了，平静地补了一句：「也可以写成对不合理称呼和定位方式有明确识别能力。」

记录员的笔停住。

陆野觉得这一刻比补贴翻倍还爽。

代价到来的时候，屏幕上只弹了一行通知。

协同C级观察对象需接受监管分离测试。

陆野往下划。

测试目的：确认能力来源、风险归属、是否存在异常兽依赖。测试期间协同C级对象需在隔离条件下完成独立路线判断。记录结果将由保守派监督存档。

陆野的手指停在屏幕上。

因为他怕测试不对。因为「记录结果由保守派监督存档」这句话，像把钥匙交给一个已经证明会锁门的人。

煤球也看见了。

它没有拍桌，没有嘶声，甚至没有炸毛。它只是伸出爪子，把新猫包的编号牌慢慢推到了手机屏幕旁边。

编号牌：XH-CAT-C-临江-001。

屏幕：监管分离测试通知。

一个代表待遇，一个代表限制。

煤球把两样东西并排放在一起，然后收回爪子，尾巴搭在桌沿，不动了。

像在等人类自己看明白。

陆野看了三秒。然后拿起新证件，把照片那面翻过去，只露出背面的条形码。

「测就测吧，」他说，「至少热水修了。」

煤球的尾巴尖在桌沿上敲了一下。不赞许，也不反对。更像一位裁判在记录比分。
